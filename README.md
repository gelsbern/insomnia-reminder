## Requirements

**Required**
- Fabric Loader
- Fabric API

**Optional**
- Cloth Config (for configuration UI)
- Mod Menu (for easy access to config screen)

> The mod will run without optional dependencies; they only enhance configuration access.

---

## Configuration

All settings are configurable in-game when **Mod Menu** and **Cloth Config** are installed.

Without these mods, default settings will still apply and the mod will function normally.

## Installation

1. Install **Fabric Loader** for your Minecraft version
2. Install **Fabric API**
3. (Optional) Install **Mod Menu** and **Cloth Config**
4. Drop the `insomnia-reminder-*.jar` into your `mods` folder
5. Launch the game
